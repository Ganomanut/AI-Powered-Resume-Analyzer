AI-Powered Resume Analyzer: Installation Guide

  1. Download the Project Files:
    Clone the GitHub repository or download the zipped project folder and extract it.
  2. Install XAMPP (if not already installed):
    Download and install XAMPP to use Apache and MySQL.
  3. Move the Project to XAMPP's htdocs:
    Place the entire project folder inside C:/xampp/htdocs/.
  4. Start Apache and MySQL:
    Open the XAMPP Control Panel and start both Apache and MySQL services.
  5. Set Up the Database:
    • Open http://localhost/phpmyadmin in your browser.
    • Create a new database (e.g., resume_analyzer).
    • Import the provided SQL file (e.g., resume_analyzer.sql) to create the necessary tables.
  6. Open the Web App:
    Visit http://localhost/your_project_folder/index.html in your browser to launch the application.
  7. Use the Resume Analyzer:
    Upload resumes, analyze job matching, and view results directly through the web interface.

This guide enables you to install and run the AI-Powered Resume Analyzer on your local environment using XAMPP and a browser.
